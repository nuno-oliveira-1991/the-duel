This is a Three.js duel shooter game app, with Cannon.js 3D physics and Yuka AI engine.
It is still in it's first stages of development, being developed as a personal project by Nuno Oliveira.

This project has been converted to React + Vite + TypeScript with @react-three/fiber, @react-three/drei, and @react-three/cannon.

To run this app:

  1. clone this repository or download it.
  2. install node packages (npm install)
  3. run the dev server (npm run dev)
  4. Have fun!!

Controls:
- W/A/S/D: Move the green box (directional movement based on rotation)
- Arrow Left/Right: Rotate the green box
- Space: Shoot projectile (fires in the direction the box is facing)
